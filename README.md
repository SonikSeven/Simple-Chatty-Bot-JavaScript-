# Simple Chatty Bot

Welcome to the Simple Chatty Bot GitHub repository! This project is a simplistic yet engaging console application built with TypeScript, designed to interact with users through a series of prompts and responses. Let’s dive into more about this project, how you can get it up and running on your machine, and the different interactions you can have with Friendly Bot.

## Project Overview

Friendly Bot is a console application created to perform a series of basic interactions with users. Built using TypeScript, it showcases fundamental programming concepts such as functions, loops, conditionals, and user input handling. The bot can greet you, guess your age based on a simple arithmetic trick, prove it can count to any number you choose, and even test your programming knowledge with a little quiz!

## Features

- **Personalized Greetings:** Friendly Bot begins the interaction by asking for your name and then greets you personally.
- **Age Guessing:** By providing remainders of your age when divided by 3, 5, and 7, the bot can guess your age.
- **Counting Ability:** The bot can count up to any number you specify, showcasing its loop skills.
- **Programming Quiz:** Test your knowledge with a simple programming-related question.
- **Warm Farewell:** Ends the conversation on a positive note.

## Requirements

- [Node.js](https://nodejs.org/)

## Installation

This application is written in TypeScript, so you'll need Node.js installed on your computer to run it. If you don't have Node.js installed, you can download it from [nodejs.org](https://nodejs.org/).

To install this project, follow these steps:

1. Clone the repository to your local machine:

```
git clone https://github.com/SonikSeven/simple-chatty-bot-typescript.git
```

2. Navigate to the cloned repository:

```
cd simple-chatty-bot-typescript
```

3. Install the required dependencies:

```
npm install
```

## How to Run

To run the program, follow these steps:

1. Open a terminal and navigate to the directory where the script is located.
2. Run the following command:

```
npx ts-node index.ts
```

## Usage

The bot will take you through a series of interactions where you can input your responses directly in the console. Follow the bot's prompts, and enjoy the conversation!
